# PrimateHumeriPaper

This project contains all residual anayltical data and code to reproduce the statistics 
and figures of the paper "Beam Bending is violated in long bones: variation in 
cortical bone geometry in primates and its ecological and biomechanical inferences"

Data are based on outputs from morphomap by Profico. 

All STL data can be requested from the Author, or accessed from Morphosource
